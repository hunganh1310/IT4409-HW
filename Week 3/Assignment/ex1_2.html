Web Technologies and Services (IT4409)

Semester 20251, Hanoi University of Science and Technology. This course introduces modern web technologies, web programming, and services to equip students with both theoretical foundations and practical skills.

Course Overview

This course covers the fundamentals of web technologies including HTML, CSS, JavaScript, server-side programming, REST APIs, and cloud services. Students will practice building responsive, secure, and scalable web applications.
